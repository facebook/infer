Type of a Datalog fact. In all facts, "proc_name" is the procedure inside of which the fact is emitted. The allocation and call sites are uniquely identified by the string "class:line:assigned_variable" (note that in SIL there is always an assigned variable, even for procedures with a void return type). The assigned variable is used instead of the column because the Java frontend does not provide information about columns, and variables have unique names.
