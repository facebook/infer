val large_step : call_location:IBase.Location.t -> callee_proc_name:IR.Procname.t -> substitution:(value * PulseValueHistory.t) PulseAbstractValue.Map.t -> keep:PulseAbstractValue.Set.t -> get_dynamic_type:( value -> IR.Typ.t option ) -> path_condition:PulsePathCondition.t -> callee_prepost:state -> state -> state

large_step ~substitution ~keep ~condition state ~callee_prepost updates state according to callee_prepost. The abstract values in condition and state are in one scope, and those in callee_prepost in another scope: the substitution maps from the callee scope to the condition&state scope.
