"resource leak" checker for .NET.

Activate with `--dotnet-resource-leak`.

Supported languages:
- C/C++/ObjC: No
- C#/.Net: Yes
- Erlang: No
- Java: No



## List of Issue Types

The following issue types are reported by this checker:
- [DOTNET_RESOURCE_LEAK](/docs/next/all-issue-types#dotnet_resource_leak)
