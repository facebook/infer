Detection of dead stores and unused variables.

Activate with `--liveness`.

Supported languages:
- C/C++/ObjC: Yes
- Java: No
- C#/.Net: No



## List of Issue Types

The following issue types are reported by this checker:
- [DEAD_STORE](/docs/1.1.0/all-issue-types#dead_store)
