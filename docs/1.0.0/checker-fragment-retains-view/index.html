Fragment Retains View

Detects when Android fragments are not explicitly nullified before becoming unreachable.

***DEPRECATED*** Unmaintained due to poor precision.

Activate with --fragment-retains-view.

Supported languages:

  • C/C++/ObjC: No
  • Java: Yes
